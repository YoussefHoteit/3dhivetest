# 3Dhive

A conversion-focused storefront for original, small-batch 3D-printed figurines.

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production checks

```bash
npm run lint
npm run build
```

## Deploy on Vercel

Import this GitHub repository into Vercel. Vercel detects Next.js automatically, so no custom build settings or environment variables are required for the current demo storefront.

The cart and checkout are currently a front-end demonstration. Connect a payment provider and order database before accepting real purchases.
